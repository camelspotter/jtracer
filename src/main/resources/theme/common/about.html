jTracer 1.0.4

Stack Trace Visualization Tool

jTracer visualizes C++ exception (or thread) stack traces generated by libcsdbg. It implements the server side of LDP (Libcsdbg Debug Protocol). It is essential for cross-debugging on target platforms which can't visualize output.
